Introduction

Welcome to this guide on how to get started on your first Unity game.

What to install

Before you can start working on your game, you have to install Unity.

Setting up files
  • Create a 2d or 3d game.
  • Create empty game objects.
  • One is for 'Game logic'.
  • One is for 'Environment'. This is where you put all physical objects.
  • One is for 'Audio Manager'.
Starter Code

Code that runs first.

void private Awake()

Code that runs after awake.

void private Start()

Code that runs every frame.

void private Update()

Code that runs as often as you define in settings.

void private FixedUpdate()

Code that runs when you hit a trigger.

void private OnTrigger(GameObject obj)
Where to upload

You can upload your game to Itch.io.

When you build your game, you can choose between making it playable on the web, or online.

Itch.io lets you upload multiple files. You can upload both a downloadable version, and an online version.
